# الإصدار المستقل (Legacy)

هذا المجلّد يحتفظ بالنسخة الأصلية من التطبيق قبل إعادة هندسته إلى مشروع
React + TypeScript. الملفات هنا **مرجعية** ولا تدخل في عملية البناء.

| الملف                  | الوصف                                                             |
| ---------------------- | ---------------------------------------------------------------- |
| `standalone.html`      | أحدث نسخة مستقلة — مزامنة سحابية مفتوحة (بدون رمز PIN).            |
| `standalone-pin.html`  | نسخة سابقة محمية برمز PIN عبر دوال RPC (`tracker_save` بوسيط PIN). |

كل ملف يعمل بمفرده بفتحه مباشرة في المتصفح — لا يحتاج إلى أي أدوات بناء أو تثبيت.

يعتمد الإصدار الحديث (في `src/`) على نفس عقد الواجهة الخلفية للنسخة المفتوحة
(`standalone.html`): `tracker_save({ p_id, p_data })` و `tracker_load({ p_id })`.
